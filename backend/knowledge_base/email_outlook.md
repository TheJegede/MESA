# Email & Outlook — Account Access Issues

## Overview
Mines provides Microsoft Exchange email to all students, faculty, and staff. Your email address is username@mines.edu. Access via Outlook desktop, web (outlook.office.com), or the Outlook mobile app.

## Common Issues & Solutions

### Cannot log in to email / Outlook Web
1. Go to outlook.office.com and sign in with your full mines.edu address.
2. If redirected to a login error, reset your MultiPass password (see Password Reset article).
3. Clear browser cookies for microsoft.com and outlook.com, then retry.
4. Complete Duo MFA if prompted — email requires MFA for web access.

### Outlook desktop app not syncing
1. Check your internet connection.
2. In Outlook, look for "Disconnected" or "Trying to connect" in the status bar.
3. File → Office Account → Sign Out, then Sign In again with your mines.edu account.
4. If Outlook prompts for a password in a loop: open Credential Manager (Windows) → remove all Microsoft/Office credentials → restart Outlook.

### Not receiving emails
1. Check your Junk/Spam folder — Outlook's spam filter sometimes misclassifies legitimate emails.
2. Check your inbox rules: Settings → View all Outlook settings → Rules — a rule may be redirecting or deleting messages.
3. Check if your mailbox is full: Settings → General → Storage. Mines provides 50GB per mailbox.
4. If expecting a specific email, ask the sender to resend and check for delivery failure notices on their end.

### Email marked as spam or bouncing
- Mines' email security scans all outbound messages. Bulk email from your mines.edu account may be flagged.
- If you need to send bulk announcements, use the approved mailing list service through IT.
- For legitimate emails being blocked by recipients: submit a ticket with the recipient domain for IT to investigate.

### Setting up email on mobile
1. Install Outlook mobile (iOS/Android).
2. Add account → use your mines.edu address.
3. You will be redirected to Mines SSO — sign in with MultiPass.
4. Complete Duo MFA enrollment if prompted.
5. Do not use the native iOS/Android mail app with Exchange — it does not support Mines' MFA requirements.

### Email quota full
- Default mailbox size: 50GB.
- Archive old emails: File → Cleanup Tools → Archive (Outlook desktop).
- Delete large attachments: Sort inbox by size, delete emails with large attachments you no longer need.
- Empty Deleted Items and Junk folders regularly.

## When to Escalate
- Cannot access email after password reset.
- Mailbox appears empty (possible data loss).
- Receiving excessive spam that spam filters are not catching.
- Need a shared mailbox or distribution list set up for a department.
