# Software Licensing — Campus Software Access

## Overview
Mines provides free or discounted software to students, faculty, and staff through institutional licenses. Software is available through the Mines Software Portal at software.mines.edu.

## Available Software

### Microsoft 365 (Office Suite)
- All students, faculty, and staff receive Microsoft 365 for free.
- Includes: Word, Excel, PowerPoint, Teams, OneDrive, Outlook.
- Install: office.com → sign in with mines.edu → Install Office.
- Up to 5 devices (PC, Mac, tablet, phone).

### MATLAB
- Free for all currently enrolled students and active faculty/staff.
- Download from software.mines.edu → MATLAB.
- Use your mines.edu email to create a MathWorks account and link to the campus license.
- If license shows as expired, your MathWorks account may need to be re-linked — contact IT.

### Adobe Creative Cloud
- Available to specific departments and programs — not a campus-wide license.
- Contact your department administrator or IT to check eligibility.
- Students in Engineering, Computer Science, and Design programs typically have access.

### ESRI ArcGIS
- Available for Geology, Civil Engineering, and related programs.
- Request access through the IT portal with your faculty supervisor's approval.

### Antivirus — Crowdstrike Falcon
- Free for all Mines devices (personal and university-owned).
- Download from software.mines.edu → Security → Crowdstrike Falcon.
- Required on any device that accesses Mines network resources remotely.

## Common Issues

### Software shows license expired
1. Sign out of the application and sign back in with your mines.edu credentials.
2. Ensure your Mines enrollment or employment is still active.
3. For Microsoft 365: go to office.com → Account → Subscription — should show active.
4. If license expired mid-term for active students: submit a ticket — IT will re-link your account.

### Cannot find software in the portal
- The software portal lists currently licensed titles. If a title is missing, Mines may not have a license.
- Submit a software request through the IT portal if your course requires software not in the portal.
- Faculty can request evaluation licenses for academic use.

### Installation fails
- Ensure you have admin rights on your device for installation.
- On Mines-managed lab computers, software must be installed by IT — submit a ticket.
- Check minimum system requirements for the software version.

## When to Escalate
- License verification fails after sign-out/sign-in.
- Software required for a class is unavailable.
- Installation fails on a Mines-managed device.
